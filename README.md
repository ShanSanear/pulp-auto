# Pulp Cthulhu — Full Auto Calculator

A browser-based calculator for resolving automatic weapon fire in **Call of Cthulhu 7th Edition / Pulp Cthulhu**. Create investigator profiles with their firearms skills and weapon loadouts, then configure volleys against one or more targets — including armor, range difficulty, and traversal costs — and let the app roll the dice and tally up hits, impales, and net damage in one click.

All data (investigator profiles) is stored entirely in your browser's `localStorage`. There is no server, no account, and no network required after the initial page load — the app runs 100% client-side.

> **This is a 100% vibe-coded app.** Every line of code — from the dice engine to the UI components — was generated through AI-assisted, prompt-driven development without any manual coding.
